# laboratorio-secops

Laboratorio de SecOps - Git, Python, LocalStack


## Descrição 

Este repositorio comtem os exercicios e projetos que estou a desenvolver no meu estagio de SecOps. O objetivo é consolidar conhecimentos em:

- **Git & GitHub** - controlo de versões, branches, pull requests, resoluções de conflictos
- **Python** - scripts de autoação, analises de logs, monitorização de sistemas
- **LocalStack** - simulação de serviços AWS localmente (S3, Lambda, IAM)
- **Projeto Integrador** - sistema de monitorização e auditoria de segurança

## Tecnologias Utilizadas

- **Linux** - Rocky Linux 8
- **Git & GitHub** - controlo de versões
- **Python 3** - scripts de autmação
- **Docker / LocalStack** - emulação de serviços cloud
